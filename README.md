# WTWR (What to Wear?)

## Project Description

WTWR (What to Wear?) is a responsive React application that recommends clothing based on the current weather. The application retrieves weather data from an external API and displays clothing items appropriate for the current temperature and weather conditions. Users can browse available clothing items, add new garments, and interact with a clean, responsive interface built with reusable React components.

## Features

- Display current weather information
- Filter clothing recommendations based on temperature
- Add new clothing items through a modal form
- Responsive design for desktop and mobile devices
- Reusable React components
- Controlled forms with React state
- Dynamic rendering using props and state
- Modal windows for user interactions

## Technologies Used

## Installation

Clone the repository:

git clone <https://github.com/erikjmitchell/se_project_react>

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
